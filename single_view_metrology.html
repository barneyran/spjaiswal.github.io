Single View Metrology

Summary

The objective of this project is to build a 3D model of flat objects from a single view.Single View Metrology demo is based on a paper by Criminisi. et al [1]. This project is a part of COMP-5421 (Computer Vision Course) at HKUST taught by Dr. C. K. Tang during the spring term (Feb-May'2014).

Input image

Texture image

YZ plane texture map

XZ plane texture map

XY plane texture map

Once we have a texture map of the normal planes of the object, we can crop the required portion of the image and create a 3D model using tools like Blender.

References

[1] Criminisi, Antonio, Ian Reid, and Andrew Zisserman. “Single view metrology.”International Journal of Computer Vision 40.2 (2000): 123-148
